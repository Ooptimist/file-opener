# 📁 文件批量打开工具

## 核心功能

**1. 批量打开文件**

- 一次选择多个文件，一键全部打开
- 自动识别文件类型并显示对应图标

**2. 文件组管理**

- 将常用文件保存为分组（如"工作文档"、"项目资料"）
- 一键打开整个文件组
- 文件存在状态实时检测（✅存在/❌丢失）

**3. 快捷操作**
- 勾选文件批量移除
- 展开文件组查看详细列表
- 数据自动保存，重启软件不丢失

---

## 界面布局

| 左侧：文件选择 | 右侧：文件组管理 |
|--------------|----------------|
| 📝 选择文件 | 📂 我的文件组 |
| 💾 保存文件组 | ▶ 展开/折叠列表 |
| 文件列表（带复选框）| 打开/删除按钮 |
| 🗑️ 移除 🚀 打开 | 文件计数（3/5） |

---

## 使用步骤

### 添加文件
点击"📝 选择文件"，多选后确认  

### 打开文件
- 点击"🚀 打开"：打开左侧所有文件
- 点击文件组"打开"按钮：打开该组所有文件

### 保存文件组
1. 左侧添加需要的文件
2. 点击"💾 保存文件组"
3. 输入组名（如"日常工作"）

### 管理文件组
- **展开**：点击 ▶ 查看组内文件
- **删除**：点击"删除"按钮移除整个组（不删除实际文件）
- **检测**：❌ 标记表示文件已被移动或删除

---

## 数据说明

- 文件组保存在软件目录的 `file_groups.json` 中
- 删除该文件会清空所有分组数据
- 支持 Windows 10/11
